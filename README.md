# Railway Management System DBMS Project

## Project Overview
This repository contains a DBMS mini project for a Railway Management System.
The project includes:
- SQL schema and sample operations
- Project report documentation
- ER diagram image

## File Analysis
1. `railway_management_system.sql`
- Main implementation file.
- Creates database `RailwayManagementDB`.
- Defines core tables: `passenger`, `station`, `train`, `route`, `schedule`, `ticket`, `payment`.
- Includes constraints such as primary keys, foreign keys, unique keys, and check constraints.
- Contains booking, schedule, and payment related structure for railway operations.

2. `project report.md`
- Full written report of the project.
- Covers abstract, objectives, problem statement, scope, methodology.
- Documents ER model, schema design logic, and SQL feature usage.

3. `ER Digram.png`
- ER diagram image for the project database.
- Visual representation of entities and relationships used in the SQL schema.

4. `README.md`
- Repository guide file.
- Contains project summary, folder structure, and file-level analysis.
